# Teste de API – Módulo 13 | EBAC

- Este repositório contém a collection de testes de API desenvolvida no **Módulo 13 do curso de QA da EBAC**.
- Os testes foram criados no **Postman**, seguindo o enunciado do exercício, com cenários positivos e negativos.


## Ferramentas utilizadas
- Postman
- API Hub de Leitura (fornecida pela EBAC)


## O que foi testado
Foram criadas requisições utilizando os principais métodos HTTP:

- GET
- POST
- PUT
- DELETE

As requisições estão organizadas em uma collection única.

## Cenários implementados
- Cenários positivos para todas as operações
- Cenários negativos (ex.: dados inválidos, ausência de token, IDs inexistentes)
- Validação de status code
- Validação de mensagens e regras de negócio


## Como executar
1. Importar a collection no Postman
2. Selecionar o environment configurado
3. Executar as requisições individualmente ou via Runner
